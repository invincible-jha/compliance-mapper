# Contributing

Please read our contribution guidelines before opening a pull request.

Full guide: https://github.com/muveraai/aumos-oss/blob/main/CONTRIBUTING.md

## Quick Rules

- All source files must include the SPDX header (`BSL-1.1`) and copyright notice.
- TypeScript: strict mode, no `any`, named exports only.
- Python: type hints on every function signature, Python 3.10+.
- Conventional commits: `feat:`, `fix:`, `refactor:`, `docs:`, `test:`, `chore:`.
- Do not add vertical-specific or jurisdiction-specific regulatory frameworks — see [FIRE_LINE.md](./FIRE_LINE.md).
